# Smart Agriculture Advisory System

## Project Overview

The Smart Agriculture Advisory System is a web-based application developed using Python and Flask. It helps farmers make informed decisions by analyzing agricultural data and providing recommendations for crop management.

The system predicts crop yield, evaluates pest risk, performs farm data quality assessment, and generates crop recommendations using machine learning models.

---

## Features

- Crop Yield Prediction
- Pest Risk Assessment
- Farm Data Quality Assessment
- Crop Recommendation Engine
- Sensor Data Analysis
- SQLite Database Integration
- Interactive Web Interface
